## Summary

This is a small, self-contained Snake game built with Pygame. The repository contains the original script (`snake.py`), a cleaned and runnable version (`snake_fixed.py`), a backup of the original (`snake_backup.py`), and CI to validate the code.

Tech: Python, Pygame, GitHub Actions

## Features

- Player-controlled snake with arrow keys
- Food spawning and scoring
- Collision detection (self and wall)
- Cleaned/refactored code for readability

## Requirements

- Python 3.11 or newer
- pygame (see `requirements.txt`)

Install requirements:

```bash
python -m pip install -r requirements.txt
```
